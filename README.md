## Vokadash | Dashboard boilerplate

build with

- Vite.js
- React
- Typescript
- Shadcn
- Tanstack Query (React)
- React Router v6

### Getting started

First, must installed via package

```
// npm
npm install @itokun99/vokadash --save

// yarn
yarn add @itokun99/vokadash

// pnpm
pnpm add @itokun99/vokadash
```

see example at dev/index.tsx
